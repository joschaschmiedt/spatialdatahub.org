About

The Leibniz Zentrum für Marine Tropenökologie Web GIS Portal is a project funded by ??????????. Our goal is to be the world's top tropical Geographic Information Systems (GIS) data displaying website. We plan to keep the code behind the website as simple as possible, while staying up to date with the latest trends (whether these two goals are mutually exclusive or not is unknown).

If you have any questions about the website, please email Patrick Curry at patrick.curry@leibniz-zmt.de.

This website has been built with Django, PostgreSQL, Bootstrap 4, and Leafletjs.

The Source Code available at: https://github.com/patzmt/WebGIS
