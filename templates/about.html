The goal of the Web GIS Portal is to be the world's top tropical Geographic Information Systems (GIS) data displaying website. It is based out of the Leibniz Zentrum für Marine Tropenökologie in Bremen, Germany. We plan to keep the code behind the website as simple as possible, while staying up to date with the latest trends. Further functionality will be added as the project matures.

Through out developement we are adhering to the Test Driven Development (TDD) method as much as possible. The TDD method requires that developers write tests for their functions before writing their functions. For example if we were creating a simple add function we would write the test that called the function and asserted an answer before the function was written. Then, after seeing the test fail, we would make it pass by writing the function. As an example have made our JavaScript testing suite available here: JavaScript testing suite.

If you would like to see the source code for the website visit our github page.

If you have any questions or comments regarding the website, please email Patrick Curry at patrick.curry(at)leibniz-zmt.de.

This website is being developed with Django, PostgreSQL, Bootstrap 4, and Leafletjs.
